# Excel Sales & Customer Performance Analysis

## Objective
Analyze sales and customer data using Excel to identify revenue trends, top customers, and key business insights.

## Tools Used
- Microsoft Excel

## Dataset
Sales transaction data including order dates, customers, cities, products, quantities, prices, and revenue.

## Key Questions
- Who are the top revenue-generating customers?
- Which cities generate the most revenue?
- What are the monthly sales trends?
- What is the average order value?
